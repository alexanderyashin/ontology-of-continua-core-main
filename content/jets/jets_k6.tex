\section{Jets on $K_6$}
\label{sec:jets-k6}

The continuum $K_6$ introduces cognitive organisation: patterns of neural
activity become structured representations, predictive models, and
memory-stabilised attractors. Jets formalise the infinitesimal
variations of these cognitive fields, the dynamics of prediction error,
the stability of representation cycles, and the thresholds of model
coherence.


% ================================================================
\subsection{Cognitive Coordinates and Jet Space of $K_6$}
% ================================================================

A state of $K_6$ is given by the tuple
\[
\omega_6 =
(R,\ E,\ M,\ W,\ \Pi,\ \Lambda),
\]
where:
\begin{itemize}
    \item $R$ — space of representations (activation patterns),
    \item $E$ — prediction error fields,
    \item $M$ — model state (internal generative model),
    \item $W$ — synaptic / mapping weights (learned transformations),
    \item $\Pi$ — attention / selection potentials,
    \item $\Lambda$ — memory traces governing longer-scale stability.
\end{itemize}

Jets describe infinitesimal variations:
\[
J^{m,n}(\omega_6)
=
\big\{
\partial_t^k\partial_{A_6}^\ell (R,E,M,W,\Pi,\Lambda)
\big\},
\quad 0\le k\le m,\ 0\le \ell\le n.
\]


% ================================================================
\subsection{Jets of Representations $R$}
% ================================================================

Representations evolve as attractor-guided activation patterns:
\[
\dot{R}=F_R(R,M,\Pi,J_{\mathrm{info}},\eta_{\mathrm{noise}}).
\]

Jets:
\[
J^1(R)=\{\dot{R},\ \partial_{A_6}R\}.
\]

These describe:

\begin{itemize}
    \item stability of attractor basins,
    \item transitions between representational states,
    \item sensitivity to attention potentials $\Pi$,
    \item deformations caused by prediction error.
\end{itemize}


% ================================================================
\subsection{Jets of Prediction Error $E$}
% ================================================================

Prediction error:
\[
E = \mathrm{sensory} - \mathrm{model}(R,M).
\]

Temporal jet:
\[
\dot{E} = F_E(E,M,R,W).
\]

Axis jet:
\[
\partial_{A_6}E =
\partial_{R}E\,\partial_{A_6}R
+
\partial_{M}E\,\partial_{A_6}M.
\]

Jets determine:

\begin{itemize}
    \item onset of learning updates,
    \item divergence of representational states,
    \item collapse of unstable models,
    \item switching between cognitive cycles.
\end{itemize}


% ================================================================
\subsection{Jets of the Generative Model $M$}
% ================================================================

The internal model evolves via:
\[
\dot{M}=F_M(M,W,E,\Lambda).
\]

Jet structure:
\[
J^1(M)=\{\dot{M},\ \partial_{A_6}M\}.
\]

Model jets govern:

\begin{itemize}
    \item coherence of world-model hypotheses,
    \item sensitivity to prediction error,
    \item transitions between competing models,
    \item stability near the model-threshold $\Theta_{\mathrm{model}}$.
\end{itemize}


% ================================================================
\subsection{Jets of Synaptic / Mapping Weights $W$}
% ================================================================

Weight update rule:
\[
\dot{W}=F_W(W,E,R,M),
\]
where updates occur only if error exceeds a learning threshold:
\[
|E|>\Theta_{\mathrm{learn}}.
\]

Jets:
\[
J^1(W)=\{\dot{W},\ \partial_{A_6}W\}.
\]

They capture:

\begin{itemize}
    \item local plasticity,
    \item formation of stable mappings,
    \item catastrophic forgetting vs.\ stabilisation by $\Lambda$,
    \item emergence of compositional representations.
\end{itemize}


% ================================================================

\Pi = P_{\mathrm{sel}}(R,M,E,\mathrm{task}),
\]

Jets:
\[
J^1(\Pi)=\{\dot{\Pi},\ \partial_{A_6}\Pi\}.

These derivatives encode:

\begin{itemize}

\end{itemize}


% ================================================================

J^1(C_j)=
